# 4. Agent 1: Text Preparation Agent

## Rolul agentului

Agentul 1 primește textul brut și îl curăță.

El face:
- transformare în lowercase;
- eliminare linkuri;
- eliminare caractere speciale;
- eliminare spații multiple;
- eliminare texte goale.

## Cod pentru src/preprocessing.py

Creează fișierul src/preprocessing.py și pune în el:

import re

def clean_text(text):
    text = str(text).lower()
    text = re.sub(r"http\S+|www\S+", "", text)
    text = re.sub(r"[^a-zA-Z\s]", " ", text)
    text = re.sub(r"\s+", " ", text).strip()
    return text

## Cod în notebook

from src.preprocessing import clean_text

df["clean_content"] = df["content"].apply(clean_text)

## Verificare

print(df[["content", "clean_content", "label"]].head())
