# Task-uri Persoana 2
## Proiect: Multi-Agent Fake News Detection using DistilBERT Features

Rol principal:
Persoana 2 se ocupă de:
- Agentul 2: DistilBERT Feature Extraction Agent
- Agentul 3: Fake News Classification Agent

Scopul părții tale:
Să transformi textele curățate în vectori semantici folosind un model de deep learning / Transformer, apoi să antrenezi un classifier care decide dacă o știre este FAKE sau REAL.

Noua variantă folosește DistilBERT pentru Agentul 2, în loc de TF-IDF sau SentenceTransformer simplu.

---

## Arhitectura proiectului

Sistemul are 3 agenți:

1. Agent 1: Rule-based Text Preparation Agent
   - făcut de cealaltă persoană;
   - curăță textul;
   - produce fișierul processed_news.csv.

2. Agent 2: DistilBERT Feature Extraction Agent
   - făcut de tine;
   - folosește modelul preantrenat distilbert-base-uncased;
   - transformă fiecare articol într-un vector numeric semantic.

3. Agent 3: Fake News Classification Agent
   - făcut de tine;
   - primește vectorii generați de DistilBERT;
   - clasifică articolele ca FAKE sau REAL.

Schema:

Raw article
    ↓
Agent 1: Rule-based preprocessing
    ↓
clean_content
    ↓
Agent 2: DistilBERT feature extraction
    ↓
BERT feature vectors
    ↓
Agent 3: Logistic Regression / MLPClassifier
    ↓
Final decision: FAKE / REAL

# 5. Funcția pentru extragerea feature-urilor cu DistilBERT

## Ce trebuie să facă funcția

Funcția primește o listă de texte și returnează un vector pentru fiecare text.

Important:
Modelele BERT au limită de lungime. Folosim:
- truncation=True
- max_length=512

Fiecare articol va fi reprezentat prin vectorul primului token din output-ul DistilBERT.

## Cod orientativ

```python
def extract_bert_features(texts, batch_size=16, max_length=512):
    features = []

    texts = list(texts)

    with torch.no_grad():
        for i in tqdm(range(0, len(texts), batch_size)):
            batch_texts = texts[i:i + batch_size]

            encoded = tokenizer(
                batch_texts,
                padding=True,
                truncation=True,
                max_length=max_length,
                return_tensors="pt"
            )

            encoded = {
                key: value.to(device)
                for key, value in encoded.items()
            }

            outputs = bert_model(**encoded)

            # Reprezentarea documentului: vectorul primului token
            batch_features = outputs.last_hidden_state[:, 0, :]

            features.append(batch_features.cpu().numpy())

    return np.vstack(features)
```

## Ce înseamnă outputs.last_hidden_state[:, 0, :]

DistilBERT produce un vector pentru fiecare token din text.
Noi luăm vectorul primului token ca reprezentare globală a articolului.

Pentru DistilBERT, dimensiunea vectorului este 768.

# 14. Ce scrii în notebook la explicații

## Pentru Agentul 2

Agentul 2 este un agent bazat pe deep learning, responsabil pentru extragerea reprezentărilor semantice ale articolelor. În loc să folosim TF-IDF, care se bazează pe frecvența cuvintelor, folosim DistilBERT, un model Transformer preantrenat.
